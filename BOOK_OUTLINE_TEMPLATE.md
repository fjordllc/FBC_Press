# FBC Press Book Outline Template

このテンプレートは、新しい書籍の全体設計を始めるときの雛形です。

**この OUTLINE が本の中身を決めます。** 既刊『Tailwind CSS 教科書』『Hotwire 教科書』の OUTLINE が 28〜40KB あるのに対し、薄くなった『HTML 雑学』の OUTLINE は 5KB のトピック箇条書きだけでした。**章ごとの議論を OUTLINE で設計しないと、執筆時に骨格を filler で埋めるしかなくなります**(`STYLE_GUIDE.md` 第0節)。トピック名を並べただけで執筆に入らないでください。

各章には、最低でも次を OUTLINE 段階で書き起こします(後述の「章設計シート」を使う)。

- その章の **主張(1文)**
- 主張を支える **具体例・対比・根拠**(積み上がる順)
- 読者が抱く **誤解** と、それを潰す場所
- 前章から受けるもの・次章へ渡すもの

## 章設計シート(全章ぶん埋める)

部・章の一覧ができたら、**章ごとに次を埋めてから執筆に入ります**。ここが埋まらない章は、まだ書ける状態にありません。トピック名だけで先に進まないこと。

```markdown
### 第X章 [タイトル]

- 主張(1文): [〇〇は△△である、なぜなら□□]
- 読者が読後にできる判断: [...]
- 積み上げる具体例 / 対比 / 根拠:
  1. [最初の例・事実]
  2. [それを踏まえて一段進む例]
  3. [実務の痛み・判断に着地する例]
- 先回りして潰す誤解: [読者が思い込みがちなこと] → [どの節で潰すか]
- 前章から受けるもの: [...]
- 次章へ渡すもの: [...]
- この章にしか書けない固有の中身: [タイトルの語を変えたら成立しない部分]
```
