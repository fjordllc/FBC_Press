# FBC Press Reference Check

この文書は、章末の参考資料を掲載する前の確認手順です。

## 確認項目

- URL が実在する
- できるだけ一次情報になっている
- 現行バージョンと整合している
- 本文で扱った論点と対応している
- 古すぎる情報を混ぜていない

## 手順

1. 章で扱った主要論点を 3〜5 個に整理する
2. 各論点に対応する一次情報を探す
3. URL を開いて、実在と内容を確認する
4. 旧バージョン情報なら、その条件が本文にも書かれているか確認する
5. 第三者記事を使う場合は、一次情報で裏取りする

## 参考資料の構成

- 公式ドキュメント
- 公式ブログまたはリリースノート
- 必要なら GitHub や作者発信
- 必要なら補助記事

## 避けること

- リンク切れのまま載せる
- 一次情報なしで第三者記事だけを並べる
- 本文で触れていない論点のリンクを大量に並べる
- 古い URL をそのまま使う
